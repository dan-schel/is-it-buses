## Getting started

Requires [NodeJS](https://nodejs.org/en) installed.

1. Clone the repo:

   ```
   git clone https://github.com/dan-schel/train-disruptions.git
   ```

2. Install dependencies:

   ```
   npm i
   ```

3. Start the server:

   ```
   npm run dev
   ```

4. Navigate to http://localhost:3000.
